
# Talk

Talk is a Web App for real-time chatting.



## Features

- Real time chatting
- Typing Indicators
- Group chat and single chat available
- Add profile picture
- Login and Logout


## Tech Stack

**Client:** MERN stack, ChakraUI

**Server:** MongoDB


## Deployment

To deploy this project run

```bash
  npm run deploy
```
